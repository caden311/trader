# Trump Trader

An AI trading bot that watches Donald Trump's Truth Social posts in real time, scores them for market-moving sentiment using Claude, and routes trades through Alpaca.

## Tech Stack

- Python
- Anthropic Claude
- Alpaca trading API
- Truth Social scraping
